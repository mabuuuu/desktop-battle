# Desktop Battle - 建筑设计文档

## 设计原则

1. **线条呈现**：所有建筑用线条绘制，与火柴人风格统一
2. **功能明确**：每栋建筑有唯一核心功能
3. **资源消耗递增**：高级建筑消耗更多资源
4. **建造需要时间**：单位在建造期间不能做其他事

## 建筑属性模型

```python
class BuildingSpec:
    name: str               # 建筑名称
    width: int              # 宽度 (px)
    height: int             # 高度 (px)
    hp: int                 # 建筑生命值
    build_cost: dict        # 建造消耗 {资源: 数量}
    build_time: float       # 建造时间 (秒)
    faction_color: str      # 阵营色
```

## 建筑列表

### 资源采集点 (Resource Node)

| 属性 | 值 |
|------|-----|
| 宽度 | 10 px |
| 高度 | 8 px |
| HP | 不可摧毁 |
| 建造消耗 | 无（开局自带） |
| 建造时间 | 无 |
| 产出 | 木材 或 矿石 |

**功能**：阵营出生时自带1个木材采集点 + 1个矿石采集点。单位靠近后自动采集。

**外观**：
- 木材采集点：绿色菱形 + 向上箭头，微微发光
- 矿石采集点：黄色菱形 + 向上箭头，微微发光

**产出速率**：每秒产出5单位资源，采集点资源无限。

### 工具台 (Workbench)

| 属性 | 值 |
|------|-----|
| 宽度 | 15 px |
| 高度 | 10 px |
| HP | 500 |
| 建造消耗 | 见下表 |
| 建造时间 | 见下表 |

**功能**：制作武器的必要设施。单位携带资源到工具台旁，选择武器进行制作。

**等级与消耗**：

| 等级 | 建造消耗 | 建造时间 | 可制作武器 |
|------|---------|---------|-----------|
| Lv1 | 木材 15 | 10秒 | 长矛、盾 |
| Lv2 | 木材 15 + 矿石 10 | 15秒 | 长矛、盾、剑、弓 |
| Lv3 | 矿石 20 + 木材 10 | 20秒 | 全部武器 |

**升级方式**：在已有工具台旁建造更高级版本，旧工具台自动拆除（资源不返还）。

**外观**：
- 水平桌面线条 + 两条桌腿
- Lv1: 简单桌面
- Lv2: 桌面 + 上方悬挂工具图标
- Lv3: 桌面 + 上方悬挂工具图标 + 发光效果

### 兵营 (Barracks)

| 属性 | 值 |
|------|-----|
| 宽度 | 20 px |
| 高度 | 15 px |
| HP | 800 |
| 建造消耗 | 木材 20 + 矿石 10 |
| 建造时间 | 20秒 |
| 生产间隔 | 30秒/单位 |
| 最大队列 | 5 |

**功能**：持续生产基础战士单位。生产需要消耗资源。

**生产消耗**：
- 每个单位：木材 5 + 矿石 3

**生产流程**：

4. 扣除资源

**外观**：
- 帐篷形状：三角形屋顶 + 矩形墙体
- 阵营色填充
- 生产中：门口有闪烁效果

## 建筑布局规则

### 建造位置约束

1. 建筑只能建造在地面（任务栏）上
2. 建筑之间最小间距 20px
3. 建筑不能重叠
4. 建筑不能建在敌方建筑旁 100px 内

### 建造流程

```
单位携带足够资源
    ↓
选择建造位置（地面，满足约束）
    ↓
开始建造（单位停留在建造位置）
    ↓
建造进度条显示
    ↓
建造完成 → 建筑出现，单位恢复空闲
```

## 建筑可被攻击

- 敌方单位可以攻击建筑
- 建筑 HP 归零时被摧毁
- 被摧毁的建筑留下废墟（5秒后消失）
- 兵营被摧毁时，生产队列取消，已生产的单位仍会完成

## 建筑与资源经济

### 初始配置

| 阵营 | 初始资源 | 初始建筑 | 初始单位 |
|------|---------|---------|---------|
| 红方 | 木材 30, 矿石 15 | 木材采集点 + 矿石采集点 | 5 战士 |
| 蓝方 | 木材 30, 矿石 15 | 木材采集点 + 矿石采集点 | 5 战士 |

### 经济循环

```
采集点 → 单位采集 → 携运回基地 → 存入阵营仓库
                                    ↓
                              建造建筑 / 制作武器 / 生产单位
```

### 阵营仓库

- 每个阵营有一个虚拟仓库，存储采集到的资源
- 仓库无容量上限
- 单位采集后需走回阵营区域（采集点附近）才算存入仓库
- 制作/建造/生产直接从仓库扣除资源
