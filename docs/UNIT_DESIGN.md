# Desktop Battle - 单位设计文档

## 单位基础属性

### 核心属性表

| 属性 | 初始值 | 说明 |
|------|--------|------|
| HP (生命值) | 1000 | 高血量保证观赏性 |
| 移动速度 | 60 px/s | 地面行走速度 |
| 攀爬速度 | 40 px/s | 攀爬窗口速度 |
| 感知范围 | 100 px | 发现敌方/资源的距离 |
| 攻击范围 | 10 px | 近战攻击距离（徒手） |
| 采集速度 | 5 资源/秒 | 在采集点采集的速率 |
| 采集携带量 | 20 资源 | 单次采集最大携带量 |
| 体型半径 | 2 px | 碰撞体半径 |

### 衍生属性

| 属性 | 计算方式 | 说明 |
|------|---------|------|
| 当前携带资源 | 0 ~ 采集携带量 | 运输中的资源量 |
| 武器 | None / Weapon实例 | 当前装备的武器 |
| 状态 | idle/walking/mining/crafting/fighting/fleeing/climbing | 当前行为状态 |
| 目标 | None / Entity引用 | 当前行为目标 |

### 属性倾向

阵营属性差异极小，仅做微调以增加趣味性，不影响公平性：

| 属性 | 红方 | 蓝方 | 差异说明 |
|------|------|------|---------|
| HP | 1000 | 1050 | 蓝方略肉 |
| 移动速度 | 65 px/s | 60 px/s | 红方略快 |
| 徒手伤害 | 3 | 2 | 红方略猛 |
| 采集速度 | 4.5/秒 | 5.5/秒 | 蓝方采集略快 |

## 单位类型

### V1 单位类型

初期只有一种基础单位，通过装备武器产生战斗差异。

| 类型 | 说明 | 生产方式 |
|------|------|---------|
| 战士 (Warrior) | 基础单位，可采集、建造、战斗 | 初始出生 / 兵营生产 |

### 后期单位类型（V2+）

| 类型 | 说明 | 生产方式 | 前置 |
|------|------|---------|------|
| 战士 (Warrior) | 基础单位 | 兵营 | 无 |
| 弓箭手 (Archer) | 远程单位 | 靶场 | 工具台Lv2 |
| 巨人 (Giant) | 高HP慢速 | 巨人营地 | 工具台Lv3 |

## 单位外观绘制

### 火柴人结构（20px高）

```
  O          ← 头: 圆, r=2, 中心(0, -18)
  |          ← 身: 线, (0,-16)→(0,-8)
 /|\         ← 臂: 左(-4,-10) 右(4,-10), 从(0,-14)出发
 / \         ← 腿: 左(-3,0) 右(3,0), 从(0,-8)出发
```

### 动画状态

| 状态 | 动画描述 | 帧数 |
|------|---------|------|
| idle | 微微呼吸，身体轻微起伏 | 2帧循环 |
| walking | 腿交替迈步，手臂摆动 | 4帧循环 |
| mining | 手臂向下挥动 | 3帧循环 |
| attacking | 手臂向前挥动（武器跟随） | 3帧 |
| climbing | 四肢交替抓握 | 4帧循环 |
| dying | 身体倒下，逐渐透明 | 6帧 |
| carrying | 一手举起资源块 | walking变体 |

### 武器绘制

武器附加在右臂末端，根据武器类型绘制不同形状：

| 武器 | 绘制方式 | 附加位置 |
|------|---------|---------|
| 徒手 | 无附加 | — |
| 长矛 | 长线 (16px) | 右手延伸 |
| 剑 | 短线 (10px) + 护手横线 | 右手延伸 |
| 盾 | 小圆 (r=2.5) | 左手 |
| 弓 | 弧线 | 左手+右手 |

## 单位生命周期

```
出生 (兵营/初始)
  ↓
idle → 选择行为
  ↓
┌─────────────────────────────────┐
│  采集 → 搬运 → 建造/制作        │  经济循环
│  巡逻 → 发现敌人 → 战斗/撤退   │  军事循环
└─────────────────────────────────┘
  ↓
HP <= 0 → 死亡动画 → 从世界移除
```

## 单位数量平衡

| 参数 | 值 | 说明 |
|------|-----|------|
| 初始单位数/阵营 | 5 | 开局即有基本劳动力 |
| 兵营生产间隔 | 30秒 | 每个兵营每30秒产1单位 |
| 最大单位数/阵营 | 30 | 防止性能问题 |
| 最大兵营数/阵营 | 3 | 限制生产速度 |
