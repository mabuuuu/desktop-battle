# Desktop Battle - 单位设计文档

## 单位基础属性

### 核心属性表

| 属性 | 初始值 | 说明 |
|------|--------|------|
| HP (生命值) | 1000 | 高血量保证观赏性 |
| 移动速度 | 红方130/蓝方120 px/s | 地面行走速度 |
| 攀爬速度 | 80 px/s | 攀爬窗口速度 |
| 感知范围 | 100 px | 发现敌方/资源的距离 |
| 攻击范围 | 10 px | 近战攻击距离（徒手） |
| 采集速度 | 5 资源/秒 | 在采集点采集的速率 |
| 采集携带量 | 20 资源 | 单次采集最大携带量 |
| 体型半径 | 2 px | 碰撞体半径 |
| 移动速度倍率 | 1.0 | 可通过设置弹窗调整 (0.5~3.0) |

### 衍生属性

| 属性 | 计算方式 | 说明 |
|------|---------|------|
| 当前携带资源 | 0 ~ 采集携带量 | 运输中的资源量 |
| 武器 | None / Weapon实例 | 当前装备的武器 |
| 状态 | 见UnitState枚举 | 当前行为状态 |
| 角色 | 见UnitRole枚举 | 当前职责角色 |
| 目标 | None / Entity引用 | 当前行为目标 |

### 属性倾向

阵营属性差异极小，仅做微调以增加趣味性，不影响公平性：

| 属性 | 红方 | 蓝方 | 差异说明 |
|------|------|------|---------|
| HP | 1000 | 1050 | 蓝方略肉 |
| 移动速度 | 130 px/s | 120 px/s | 红方略快 |
| 徒手伤害 | 3 | 2 | 红方略猛 |
| 采集速度 | 4.5/秒 | 5.5/秒 | 蓝方采集略快 |

## 单位类型

### V1 单位类型

初期只有一种基础单位，通过装备武器产生战斗差异。

| 类型 | 说明 | 生产方式 |
|------|------|---------|
| 战士 (Warrior) | 基础单位，可采集、建造、战斗 | 初始出生 / 兵营生产 |

### 后期单位类型（V2+）

| 类型 | 说明 | 生产方式 | 前置 |
|------|------|---------|------|
| 战士 (Warrior) | 基础单位 | 兵营 | 无 |
| 弓箭手 (Archer) | 远程单位 | 靶场 | 工具台Lv2 |
| 巨人 (Giant) | 高HP慢速 | 巨人营地 | 工具台Lv3 |

## 单位外观绘制

### 火柴人结构（20px高，关节式）

```
   O          ← 头: 圆, r=2, 中心(fx, fy-18)
   |          ← 身: 线, (fx, fy-15)→(fx, fy-9) (肩→髋)
  /|\         ← 臂: 肩(fx,fy-15)→肘→手 (上臂4px+前臂4px)
  / \         ← 腿: 髋(fx,fy-9)→膝→脚 (大腿5px+小腿5px)
```

关节系统:
- 上臂4px, 前臂4px (肩→肘→手)
- 大腿5px, 小腿5px (髋→膝→脚)
- idle: 手臂自然垂落(肘微弯0.2弧度), 腿自然站立(膝微弯0.15弧度)
- walking: 腿交替摆动(前摆膝弯大/后蹬膝直), 手臂反向摆动
- 动画驱动: 时间连续(anim_time), 不再使用帧计数

### 血条显示

头顶血条**不展示**。单位HP通过其他方式（如受伤闪烁）反馈。

### 角色标签

调试模式下，每个单位头顶显示角色标签:
- G=生产者, B=建造者, S=战士, R=侦察, ?=空闲

### 武器绘制

武器附加在右臂末端，根据武器类型绘制不同形状：

| 武器 | 绘制方式 | 附加位置 |
|------|---------|---------|
| 徒手 | 无附加 | — |
| 长矛 | 长线 (16px) | 右手延伸 |
| 剑 | 短线 (10px) + 护手横线 | 右手延伸 |
| 盾 | 小圆 (r=2.5) | 左手 |
| 弓 | 弧线 | 左手+右手 |

## 单位生命周期

```
出生 (兵营/初始)
  ↓
idle → 选择行为
  ↓
┌─────────────────────────────────┐
│  采集 → 搬运 → 建造/制作        │  经济循环
│  巡逻 → 发现敌人 → 战斗/撤退   │  军事循环
└─────────────────────────────────┘
  ↓
HP <= 0 → 死亡动画 → 从世界移除
```

## 单位数量平衡

| 参数 | 值 | 说明 |
|------|-----|------|
| 初始单位数/阵营 | 5 | 开局即有基本劳动力 |
| 兵营生产间隔 | 30秒 | 每个兵营每30秒产1单位 |
| 最大单位数/阵营 | 30 | 防止性能问题 |
| 最大兵营数/阵营 | 3 | 限制生产速度 |
